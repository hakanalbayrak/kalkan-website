5Privacy & Trust
Create a set of 5 iPhone app store screenshots at 1290x2796 pixels each, arranged side by side in a single wide image. Dark gradient background transitioning from #0a0118 through #1a0533 to #2d1b69. Each frame features an iPhone 15 Pro mockup in the lower 65% of the frame with a purple radial glow (rgba 139,92,246 at 0.2 opacity) emanating from behind the device. The phone has a realistic titanium frame, 56px corner radius, and thin bezel. Upper 35% of each frame is reserved for headline text area (leave blank/empty for text overlay). Background has a subtle dot grid pattern (32px spacing, purple dots at 0.06 opacity). A soft directional light from top-center creates depth. Frame 5 uses green accent (#34d399) instead of purple, featuring a shield with checkmark icon instead of phone. Minimal, premium, institutional design. Purple/violet accent colors matching #8b5cf6 brand. Absolutely no text, no words, no letters, no characters, no typography anywhere in the image.

5Privacy
Create a Google Play Store promotional image set: first, a wide landscape feature graphic at 1024x500 pixels showing a dark gradient background from #0a0118 through #1a0533 to #2d1b69 at 135 degrees. On the right side, a glowing app icon in shield shape (200x200px, rounded square with 22% corner radius) with a purple gradient from #a78bfa to #7c3aed, radiating a soft purple glow. On the left side, empty space reserved for text overlay. A subtle dot grid pattern across the background. Then, next to it, show 5 Android phone screenshots at 1080x1920 pixels each with the same dark purple gradient background. Each features a Google Pixel 8 style phone mockup (rounded rectangle, thin bezels, centered camera dot) in the lower 65% with a purple radial glow behind it. Upper 35% is blank for text overlay. Frame 5 uses green accent (#34d399) with a shield-checkmark icon instead of a phone. Premium, minimal, institutional feel. Purple/violet matching #8b5cf6. Absolutely no text, no words, no letters, no characters, no typography anywhere.
Execution

How to Build the Final Assets

1

Generate Backgrounds

Paste each DALL-E prompt into ChatGPT Plus. Download all generated images.

2

Take App Screenshots

Capture 5 screens from Kalkan app on iPhone (1290×2796) and Android device (1080×1920).

3

Composite in Photoshop

Place DALL-E backgrounds → Add device mockup frame → Insert actual app screenshot inside → Add text overlays using Plus Jakarta Sans 800.

4

Export & Upload

Export as sRGB PNG. Upload to App Store Connect and Google Play Console.
